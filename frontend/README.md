# Real Estate Broking Portal - Frontend

React-based admin portal for managing leads and users.

## Setup

```bash
npm install
npm start
```

Runs on `http://localhost:3000`

## Features

- Excel file upload
- Lead management
- User management  
- Dashboard with analytics
- Timeline visualization

## Default Admin Login

- Email: admin@realestate.com
- Password: AdminPass123!

## Environment

Make sure backend is running on `http://localhost:3001`
